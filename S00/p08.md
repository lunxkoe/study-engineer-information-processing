## UI 표준을 위한 환경 분석

- 기존/현존 UI 경향 숙지

- 사용자의 요구사항을 파악

### 기능 및 설계 분석
- 기능 조작성 분석
- 오류 방지 분석
- 최소한의 조작으로 업무 처리 가능한 형태 분석
- 최소한의 조작으로 업무 처리 가능한 형태 분석

### UI 요구사항 요소
- **데이터 요구**
- 기능 요구
- 제품, 서비스 품질
- 제약 사항

- 육하원칙에 따라서 시나리오 확인
